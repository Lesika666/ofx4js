Class BaseSellInvestmentTransaction

Base class for all investment transactions for selling securities.
This class exposes a read-only view of the flattened aggregates that are common to all sell investment transactions as a convenience to application developers who may not find the ofx aggregation model intuitive.

Jon Perlow
